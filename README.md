### 🚀 Featured Project

#### [Zero Trust Ransomware Resilience System](https://github.com/MohsanRazaq/zero-trust-ransomware-resilience)
An automated, real-time file-system monitoring daemon designed to actively detect and mitigate zero-day ransomware deployments on endpoints.

* **Multi-Factor Detection:** Correlates file modification velocity (`collections.deque` sliding time windows) with structural content analysis via a **Shannon Entropy engine** to map data-encryption anomalies.
* **Proactive Defenses:** Enforces a strict Zero Trust policy by isolating files *prior* to mutation and executing instant, automated process locks (`PID` termination) when math profiles breach thresholds ($>7.5/8.00$).
* **Forensics Generated:** Exports detailed operational triages displaying real-time cryptographic signals for incident response teams.
